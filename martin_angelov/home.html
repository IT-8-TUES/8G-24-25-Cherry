Programming languages are used to create software applications. They provide a set of instructions that a computer can understand and execute.
It is split into 3 main parts:

Front end

Front end programming languages are used to create the user interface of a website or application

Back end

Back end programming languages are used to create the server side of a website or application

Computer networking

Computer networking programming languages are used to create network protocols and applications
